# oskar-Розгортання простого PHP-файлу на хостингу

Мета:
Навчитися розміщувати прості PHP-файли на хостингу та перевіряти їхню роботу через браузер.
Загальні інструкції (для всіх варіантів):
1.	Створіть обліковий запис на обраному безкоштовному хостингу.

2.	Активуйте свій сайт — дочекайтесь створення домену (зазвичай надається безкоштовна адреса на зразок yourname.epizy.com).

3.	Встановіть FTP-клієнт (наприклад, FileZilla), введіть логін/пароль та сервер з панелі хостингу.

4.	Створіть файл index.php відповідно до вашого варіанта.

5.	Завантажте index.php у папку htdocs або public_html через FTP.

6.	Відкрийте сайт у браузері за наданим доменом та переконайтесь, що PHP-файл працює.

Варіант 1. Локалізований вивід дати та часу з cookie
Завдання:
●	Реалізуйте вибір мови (uk/en/de/fr) через форму.
●	Виводьте дату і час у форматі відповідної мови.
●	Збережіть мову в cookie, щоб при повторному заході вона автоматично застосовувалася.
●	Виведіть IP користувача.

Функції PHP: setcookie(), $_COOKIE, DateTime, strftime(), locale, setlocale()
